# GaneshShop - India's Ultimate Shopping Destination

GaneshShop is a modern, full-stack e-commerce marketplace built to provide a seamless shopping experience. Users can browse products across multiple categories, search with real-time suggestions, and manage their orders through a premium dashboard.
